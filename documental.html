1.HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear y estructurar páginas web. HTML define la estructura del contenido utilizando etiquetas y elementos que son interpretados por los navegadores para mostrar texto, imágenes, videos, formularios, etc. A continuación, te explico las principales funciones de HTML:

1. Estructurar el Contenido de una Página Web

  • HTML: proporciona la estructura básica de una página web utilizando elementos como:
  • <html>: La etiqueta raíz de cualquier documento HTML.
  • <head>: Contiene metadatos como el título de la página, enlaces a hojas de estilo y scripts.
  • <body>: Donde se coloca todo el contenido visible para el usuario, como texto, imágenes, tablas, formularios, etc.

2. Definir Encabezados y Texto

  • permite organizar el contenido en títulos y párrafos:
  • <h1> a <h6>: Definen encabezados de diferentes niveles, siendo <h1> el más importante.
  • <p>: Define párrafos de texto.
  • <b>, <strong>, <i>, <em>: Etiquetas para enfatizar o destacar texto en negrita o cursiva.

3. Incorporar Enlaces

    1 JavaScript

    JavaScript es un lenguaje de programación ampliamente utilizado para el desarrollo web. Su funcionalidad es diversa y se puede aplicar en múltiples contextos, tanto en el lado del cliente (navegador) como en el lado del servidor (con Node.js). Aquí te explico algunas de las funcionalidades más importantes de JavaScript:

    1. Manipulación del DOM (Document Object Model)

    Descrisón: JavaScript puede interactuar con el DOM, lo que permite modificar el contenido y la estructura de las páginas web de manera dinámica. Ejemplo:Cambiar el texto de un párrafo al hacer clic en un botón.

    javascript

    document.getElementById('miBoton').addEventListener('click', function() { document.getElementById('miParrafo').textContent = 'Texto cambiado!'; });

    2.Eventos

    Descripción: JavaScript permite responder a eventos como clics, movimientos del ratón, teclados, etc., lo que facilita la creación de interfaces interactivas. Ejemplo: Mostrar un mensaje cuando el usuario pasa el ratón sobre un elemento.

    un enjemplo de del codigo de javascript

    ('miElemento').addEventListener('mouseover', function() { alert('Has pasado el ratón por aquí!'); }); 3. Validación de Formularios Descripción: JavaScript se usa para validar datos de formularios en el navegador antes de que se envíen al servidor. Ejemplo: Asegurarse de que un campo de correo electrónico no esté vacío. javascript Copiar código function validarFormulario() { var email = document.getElementById('email').value; if (email === '') { alert('El campo de correo electrónico no puede estar vacío'); return false; } return true; }

    4. Asincronía y AJAXh

    : JavaScript permite realizar solicitudes HTTP a servidores sin recargar la página, lo que mejora la experiencia del usuario mediante la carga dinámica de datos. Ejemplo: Cargar datos de un servidor cuando se hace clic en un botón. javascript Copiar código fetch('https://api.ejemplo.com/datos') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error)); 5. Programación Orientada a Objetos (OOP) Descripción: JavaScript soporta la programación orientada a objetos, lo que permite crear objetos reutilizables con propiedades y métodos. Ejemplo: Crear una clase y un objeto.

    el codigo de javascript

    class Persona { constructor(nombre, edad) { this.nombre = nombre; this.edad = edad; } saludar() { console.log(`Hola, mi nombre es ${this.nombre} y tengo ${this.edad} años.`); } } const persona1 = new Persona('Carlos', 30); persona1.saludar(); // Salida: Hola, mi nombre es Carlos y tengo 30 años. 6. Manipulación de Arrays y Objetos Descripción: JavaScript tiene potentes métodos para manipular arrays y objetos, lo que permite realizar tareas como ordenar, filtrar, y transformar datos. Ejemplo: Filtrar un array para obtener números pares.

    El codiogo javascript de ejemplo

    const numeros = [1, 2, 3, 4, 5, 6]; const pares = numeros.filter(numero => numero % 2 === 0); console.log(pares); // Salida: [2, 4, 6] 7. Funciones de Alto Orden Descripción: JavaScript permite usar funciones como ciudadanos de primera clase, es decir, funciones pueden ser pasadas como argumentos, retornadas por otras funciones, y asignadas a variables. Ejemplo: Usar map para transformar un array.

    el codigo de javascript

    const numeros = [1, 2, 3, 4]; const dobles = numeros.map(num => num * 2); console.log(dobles); // Salida: [2, 4, 6, 8] 8. Compatibilidad Multiplataforma Descripción: JavaScript se ejecuta en todos los navegadores modernos, lo que lo convierte en un lenguaje muy accesible para desarrollar aplicaciones web que funcionen en diferentes dispositivos. 9. Desarrollo en el Lado del Servidor Descripción: Con Node.js, JavaScript se puede usar para desarrollar aplicaciones en el lado del servidor, como servidores web, aplicaciones de red, y más. Ejemplo: Crear un servidor web simple con Node.js.

    el codigo de javascript

    const http = require('http'); const server = http.createServer((req, res) => { res.statusCode = 200; res.setHeader('Content-Type', 'text/plain'); res.end('Hola, Mundo!\n'); }); server.listen(3000, () => { console.log('El servidor está escuchando en http://localhost:3000/'); }); 10. Compatibilidad con Frameworks y Librerías Descripción: JavaScript es la base de muchos frameworks y librerías como React, Angular, Vue.js, jQuery, que facilitan el desarrollo de aplicaciones web complejas y modernas. 11. Manipulación de Fechas y Tiempos Descripción: JavaScript incluye objetos y métodos para trabajar con fechas y horas. Ejemplo: Obtener la fecha actual.

    el codigo de javascript

    const fechaActual = new Date(); console.log(fechaActual); // Salida: muestra la fecha y hora actual. JavaScript es un lenguaje extremadamente versátil que se ha convertido en una herramienta esencial en el desarrollo de aplicaciones web modernas.
